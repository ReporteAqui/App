# App

Uma plataforma de denúncias digital para as pessoas que são impactadas com a falta de saneamento básico e informações sobre o meio ambiente. Com foco em transparência, a solução busca  reduzir a burocracia na formalização de demandas, bem como prover maior proximidade com os órgãos responsáveis e de forma a permitir agilizar as soluções.

> [!AVISO]
> Versão recomendada do [NodeJs](https://nodejs.org/en) é 24.x ou superior
